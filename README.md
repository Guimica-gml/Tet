# Tet
It's tetris, you know tetris.

## Keys

Key   | Action
----- | :-----
X     | Rotates left
C     | Rotates right
→    | Moves right
←    | Moves left
↓    | Moves down
SPACE | Jumps down

## How to build
You will need [raylib](https://www.raylib.com/) in order to build this.
```console
$ ./build.sh
$ ./tet
```
