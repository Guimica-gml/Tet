# Tet
It's tetris, you know tetris.<br>
Also it uses the [Super Rotation System](https://tetris.wiki/Super_Rotation_System).

## Keys

Key   | Action
:---- | :-----
X     | Rotates left
C     | Rotates right
→    | Moves right
←    | Moves left
↓    | Moves down
SPACE | Jumps down

## How to build
You will need [raylib](https://www.raylib.com/) in order to build this.
```console
$ ./build.sh
$ ./tet
```
